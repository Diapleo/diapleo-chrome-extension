# Diapleo — DeFi Risk Scores (Chrome Extension)

See protocol risk scores directly on Twitter/X profiles and tweets. Powered by [Diapleo](https://diapleo.com).

This repository contains the **built, unpacked extension** (Manifest V3). Source lives in the `diapleo-social` monorepo under `packages/extension` (WXT + Preact).

## Install (unpacked)

1. Download/clone this repo.
2. Open `chrome://extensions`, enable **Developer mode**.
3. Click **Load unpacked** and select this folder.
